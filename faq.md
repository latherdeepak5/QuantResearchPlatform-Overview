# FAQ

## Is this financial advice?

No. This repository is for research documentation only. It is not financial, investment, or trading advice.

## Is the source code public?

No. The source code, Pine scripts, data-processing code, and private backtesting implementation are not included in this repository.

## Can someone trade directly from this repository?

No. This repository does not contain executable strategy logic or complete trading rules.

## Why make an overview public?

The public overview makes it easier to communicate the research direction without exposing proprietary implementation details.

## Can people clone this overview repository?

Yes. Public GitHub repositories can be cloned or downloaded. That is why this repository contains only non-sensitive overview material.

## Where is the real implementation?

The working implementation is kept in a separate private repository.

## Are the results guaranteed?

No. Historical backtests and deep tests do not guarantee future performance.

## What markets are being researched?

The current focus is futures strategy research, especially MES. Other markets may be reviewed later.

## Can collaborators contribute?

Private collaboration can be considered separately. Public repository updates should remain high-level and should not include private code or exact proprietary strategy logic.
